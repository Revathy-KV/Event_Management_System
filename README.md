## ⚙️ How to Run Locally

### Prerequisites
- Node.js installed → [Download here](https://nodejs.org)

### Steps

**1. Clone the repository**
```bash
git clone https://github.com/Revathy-KV/event-feedback-management-system.git
```

**2. Go into the project folder**
```bash
cd event-feedback-management-system
```

**3. Install dependencies**
```bash
npm install
```

**4. Start the backend server**
```bash
node server.js
```

**5. Open frontend**
- Open `index.html` in your browser using Live Server
- Or just double click `index.html`

**6. Use the app**
- Go to Events page to see sample events
- Go to Feedback page to submit feedback
- View all submitted feedbacks on the same page
